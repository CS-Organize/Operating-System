# Inter-Process Communication

`File`, `Signal`, `Socket`, `Message Queue`, `Pipe`, `Shared Memory` 등 다양한 방법으로 프로세스간 통신을 할 수 있다.\
이러한 방법들은 성능, 사용법, 특징 등이 다르기 때문에 상황에 맞게 사용해야 한다.
